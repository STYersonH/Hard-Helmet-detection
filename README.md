# Hard Helmet detection
  
# Install local
- Install dependencies
```
pip install -r requirements.txt
```

- Run the applications
```
python flaskapp.py
```

# Install with Docker

- Build image
```
  docker build -t hard-helmet-detection-python .
```

- Run image
```
  docker run -p 5000:5000 hard-helmet-detection-python
```
